HOW TO APPLY?

  • Joining GitHub
    • Go to github.com. Sign Up on GitHub. Choose a username (commonly referred to as GitHub handle) for you, and voila! You‚Äôve met the first requirement of being a part of Hack_MAp.
  • Registering on Hack-MAp website
    • Go to the website of Hack_MAp . Navigate to the bottom of the page and find the form to Register. Fill in all your Details. Now, if you have created an account on GitHub, you have to name your repository same as your team name . Hoorah! You‚Äôre now in.
  • Choosing a project
    • First of all, you can choose as many projects as you want. It‚Äôs not about doing everything on just one project, but solving as many real-Life issues as you can, contributing as much as you can.
  • Writing documentation
    • The code that you have added must be documented thoroughly. Open Source is about writing code that humans can read and understand. Without documentation, it is so hard to read any code. Hence, add information in README, in the comments of code, etc.
  • Wrapping up Report Submission
    • At the end of the 43 days, you will have to create a report and share your github link to CSI JUET github page . Failing to do so, will not result in successful participation. The report can be as descriptive as you want, but must contain the following points :
      • List of projects you worked on
      • List of Pull Requests you created
      • Summary of your work

About hack_MAp


Hack_MAp is a 1 month-long Event of open source software contribution run by OSDG and DEVELOPMENT CLUB in Tachyon techfest by Computer Society of India(CSI)


Technology students ! You're invited to participate in Hack_Map 19 a hackathon at the Jaypee University Of Engineering and Technology. At this 43 days, non-stop digital product development competition, you'll be challenged to use technologies such as Web Development , IoT, Cloud Computing, Image Processing, Big Data, Artificial Intelligence, Embedded Systems,Robotics,Game Development and more to design and build an innovative digital solution to any real-life problem of your choice.
